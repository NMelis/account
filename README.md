# account

Account

## Prerequisites

You will need:

- `docker`
- `docker-compose` [version at least](https://docs.docker.com/compose/compose-file/#compose-and-docker-compatibility-matrix) `18.02`

## Start
- `./run.sh`
- open http://127.0.0.1/swagger/
- Admin panel: http://127.0.0.1/admin/ (login/pass: admin)
